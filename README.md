# Svelte Todo App

Basic todo app using svelte for education purposes, based on Frontend Mentor Challenge [Link](https://www.frontendmentor.io/challenges/todo-app-Su1_KokOW)

View [Demo](https://webmaek-svelte-todo-app.netlify.app/)

## Install

Run
```
yarn
```

## Run the server

Run
```
yarn dev
```
